### 🚀 Flagship Project: Agentic Cloud Security & Remediation Framework (ACSRF)

A graph-first cloud security architecture designed to autonomously hunt for multi-hop privilege escalation paths in AWS.

![ACSRF Graph Visualisation](https://github.com/Ojasveer955/Agentic-Cloud-Security-and-Remediation-Framework/blob/main/docs/images/Graph.png)

* 🕸️ **Graph Analysis:** Models AWS environments as attack graphs using Neo4j.
* 🤖 **Autonomous Discovery:** Utilizes a multi-turn Correlation Agent to reason through and discover complex attack vectors.
* 🔒 **Auto-Remediation:** Deploys an AI-driven agent to generate Terraform-based infrastructure fixes on the fly that don't break production.
